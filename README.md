## Installation

Install Laravel 9
```bash
composer create-project laravel/laravel sample-project
```

Go to project folder
```bash
cd sample-project
```

Install Laravel Cms:
```bash
composer require tinigin/laravel-cms
```

Run install command
```bash
php artisan cms:install
```

Setup admin user
```bash
php artisan cms:admin
```
